std::vector< int > parse_vote_ranges ( string  voteranges)

This function parses the ranges in the voteranges parameter. It takes a string like "q1:0-100,q2:6-30,q5:0-50" and returns a vector of integers like [0, 100, 6, 30, 0, 50]

Parameters

Returns
A vector of integers.
